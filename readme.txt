This project is to find the matching sentence from the inputed word in multiline text

Features
-Built in angular and used angular material
-Works when 2 (or even more) words as input and displays the lines where both the words exist  independently. 


Installation
run npm install to install all packages from package.json
run ng serve
On the browser type localhost:4200 
while providing multiline input in text area after press enter after each line otherwise it would be considered as single line 

Output
The matched line and line number would be seen in expansion panel
